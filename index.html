What is cloud computing?

• Cloud computing is on-demand access, via the internet, to computing resources—applications, servers,storage, development tools etc.

How much does the cloud cost?

• Public clouds charge on a per-use basis, so costs will vary wildly based on multiple factors.

Is the cloud secure?

• The clouds themselves are generally more secure than most private data centers.

What are cloud computing services?

• SaaS (Software-as-a-Service)
• PaaS (Platform-as-a-Service)
• IaaS (Infrastructure-as-a-Service).
